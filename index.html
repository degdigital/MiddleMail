A Framework Built For A Modern Workflow

MiddleMail is a responsive email framework built with a modern workflow in mind.
It's built on top of the Middleman static site generator and features:

  • Modular Components
  • Integrated Sass
  • Yaml Data
  • Grunt Tasks
  • Automated Testing
  • Ish
  • Project Hubs
  • ESP Add-Ons

Get Started

  1. Install Middleman $ gem install middleman
  2. Clone the MiddleMail project template $ git clone git://github.com/degdigital/MiddlemanEmailTemplate.git ~/.middleman/MiddleMail
  3. Initialize your project $ middleman init [ProjectName] --template=MiddleMail
